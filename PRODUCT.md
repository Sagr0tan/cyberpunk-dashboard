# Product

## Register

product

## Users

Personal use — single user, always-open browser tab, daily driver for productivity and information.
Primary device: desktop. Context: focused work sessions, need quick access to notes, tasks, news, and project status.

## Product Purpose

Personal cyberpunk-themed dashboard aggregating notes, todos, news, videos, and project tracking into one unified hub.
Replaces multiple scattered browser tabs. Success: open once, use all day.

## Brand Personality

Efficient. Precise. Immersive. — Retrofuturistic tool aesthetic. The UI is the tool.

## Anti-references

- Generic blue/white SaaS productivity dashboards (Notion, Linear default light theme)
- Overcrowded dark UIs with gray-on-slightly-darker-gray (poor contrast)
- Decorative cyberpunk aesthetics that sacrifice readability for atmosphere

## Design Principles

1. Data density serves utility — every pixel earns its place
2. Neon accents are functional — they mark active states, live data, interactive elements
3. Every animation has a job — feedback, state change, or attention
4. Modularity is structural — each widget self-contained, collapsible, independent
5. HUD aesthetic is the signature — angular brackets, precise borders, military-grade feel

## Accessibility & Inclusion

WCAG AA contrast minimum on all text. prefers-reduced-motion respected.
Keyboard navigation supported on all interactive elements.
